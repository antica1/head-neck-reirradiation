## 二、内置推理链

### Step 1：算前一程残余

```
前一程 EQD2 = 处方 × (1 + 2/αβ) / (1 + 2/αβ) = 处方
间隔年数 → 修复比例
  <6 月 → 不修正
  6-12 月 → 修复 30%
  1-2 年 → 修复 50%
  >2 年 → 修复 70%

累积晚期 EQD2 = 前一程残余 + 本程 EQD2
累积肿瘤 BED₁₀ = 前一程 BED₁₀（不复修）+ 本程 BED₁₀
```

### Step 2：筛选可行方案

| 方案 | 处方 | 免疫增敏 | 适用条件 |
|------|------|---------|---------|
| **Quad-Shot + IO** | 3.7 Gy × 4 BID，Q4w × 3 程 | PD-1（Pembrolizumab） | 前一程 ≤54 Gy 或间隔 >3 年 |
| **Q3w 单次 + IO** | 3.5 Gy × 1，Q3w × 6 程 | PD-1 或 ADC | 前一程 ≤66 Gy |
| **常规分割** | 2 Gy × 15-20 次 | 不可用增敏 | 所有，最安全 |
| **SBRT 连续** | 5-6 Gy × 5 | PD-1 | 前一程 ≤50 Gy |

### Step 3：裁决

```
累积晚期 EQD2 ≤40 Gy → 方案无限制
累积晚期 EQD2 40-50 Gy → Quad-Shot 可用，SBRT 连续受限
累积晚期 EQD2 50-60 Gy → 仅 Q3w 单次或常规分割
累积晚期 EQD2 >60 Gy → 仅常规分割（且 ≤15 次）
```

## 三、内置证据库

### Quad-Shot 方案（JAMA Otolaryngol 2026）

| 参数 | 值 |
|------|-----|
| 处方 | 3.7 Gy × 4 次（BID，间隔 ≥6h），每 4 周一程，最多 3 程 |
| 联合药物 | Pembrolizumab 200 mg Q3w |
| 总物理剂量 | 14.8 Gy/程 × 3 = 44.4 Gy |
| 适应人群 | 复发/不可切除/转移性 HNC |
| 免疫机制 | 连续 4 次 BID→累积 ICD 跨越免疫激活阈值；Q4w 间隙→T 细胞扩增 |
| 正常组织 | 每程 BED₃=33.1 Gy₃→修复后~20 Gy₃→3 程累积~60 Gy₃→EQD2~36 Gy |

### Q3w 单次 + IO 方案（九院经验）

| 参数 | 值 |
|------|-----|
| 处方 | 3.5 Gy × 1，每 3 周，共 6 程 |
| 联合药物 | PD-1 或 ADC（照射后 24h 内） |
| 总物理剂量 | 21 Gy |
| 机制 | 每次低剂量 RT→周期性抗原释放；3 周间隙→正常组织修复 |
| 局限性 | 单次 3.5 Gy 可能低于 ICD 最佳阈值（6-12 Gy） |
| 正常组织 | 累积晚期 EQD2~18 Gy（3 周修复修正后） |

### 免疫阈值概念

```
单次剂量 < 4 Gy   → ICD 信号可能不足以激活免疫
单次剂量 4-6 Gy   → 弱至中等 ICD
单次剂量 6-12 Gy  → 最佳 ICD 窗口
单次剂量 >12 Gy   → 可能杀伤浸润 T 细胞
```
